5、如何定义自定义时间格式  （stringToDate）

var birth = "1995-09-09 00:00:00";
方案一、
_date = new Date( birth );  

定义一个函数，功能将字符串格式的时间转成 真正的时间对象
function stringToDate( str ){
     return new Date(str);
} 

方案二、  
_date = new Date( Date.parse(birth) );
Date.parse()  获取的是从1970年1月1日 午夜到  birth  这个时间段的毫秒值


定义一个毕业时间：
var end = new Date("2017-11-30");
