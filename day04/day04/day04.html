     函数体 -- 实现某个功能
}

6、函数的参数

参数分类  ： 形参和实参
形参 和 实参的区别：

形参：
--形参出现在函数的定义中
--形参必须是变量  
实参：
--实参出现在函数的调用中
--实参可以是常量、变量、表达式，但必须有值，因为参数的传递过程 是 将实参的值传递给形参 
--参数传递时 实参和形参一一对应
--如果实参的个数大于形参的个数，多余的实参自动舍去
--如果形参的个数大于实参的个数，多余的形参值为undefined

8、函数的返回值

var num =  prompt("请输入");  方法 
通过  return 关键字 可以将一个函数的结果返回  ，返回值好处 这个结果还可以用作其它功能实现
一个函数如果有返回值，有且只有一个 
一个函数一旦遇到return ，return 后面的代码将不再执行 ，类似 循环中的break
一个函数如果没有返回值，也就是没有return 语句，   返回值默认是 undefined

9、变量的作用域

全局变量
   程序从开始到结束变量均有效  
   在函数外部定义的变量  或  在函数内部没有通过var定义的变量

局部变量
   函数体内部明确通过var声明的变量
   局部变量的作用范围 变量所在的函数体
   函数调用完毕后，局部变量会自动销毁  
   局部变量的优先权大于全局变量
   所有的形参都是局部变量

10、变量的提升

在函数体内部用var声明的变量，会将变量的声明提升到函数的最顶端，只提升声明，但不赋值

function fun(){
     alert(m);
     var m = 10;
}
等价 
function fun(){
     var m;   变量提升
     alert(m);
     m  = 10;
}
