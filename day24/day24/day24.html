2、原型

所有的构造函数都有一个原型属性prototype，这个原型属性也叫做原型对象
所有的构造函数new出来的对象都指向原型对象prototype


原型方法、属性  和  普通(实例)方法、属性  区别：

优先级 ： 实例属性和实例方法 优先级高      (css中的style)   
复用性 ： 原型方法可以复用 实例方法不可以复用（css中的class  复用   css中的style不能复用  <p style="color:red">）

3、原型链

实例对象和原型之间的链接 就叫做原型链  表示 ：  __proto__
原型模式的执行流程 ：
      
     首先在实例中查找  如果实例中没有 就去原型中查找  如果原型中还没有，就去  Object.prototype  查找 如果有 就返回

6、继承方式一  --  通过改变父类的执行环境  

只能继承实例属性和实例方法
function Father(){
     this.money = 999999;
     this.dance = function(){
          alert("跳舞");
     }
}
function Son(){

     this.parent =  Father;
     this.parent();
}
var son = new Son();

7、通过call方法继承

改变父类中的this指向  让父类中的this指向子类
在子类中通过call方法调用父类
 call方法的第一个参数 ： 要改变的Animal中的this指向
 call方法的第二个参数个数 不固定     ，就是要继承的父类的属性
 父类.call(this,父类属性);

8、通过apply继承

apply 的第一个参数 要改变的父类中的this指向
apply 的第二个参数 是一个数组    可以通过arguments代替 

9、原型继承

原型继承   ： 让父类的实例对象指向子类的原型
