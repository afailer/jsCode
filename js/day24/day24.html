2、原型  prototype

所有的构造函数都有一个prototype属性，这个属性就叫做 原型对象 
原型方法和实例方法的区别 ：
原型方法 ： 优先级低   ，但是 方法可以复用    （对比成css中的class）
实例方法（普通方法） ： 优先级高  ， 方法 不能复用    （对比成css中的style）
实例优先级高 ：
<script type="text/javascript">
    function Animal(){
       this.name = "小羊";
       this.eat = function(){
           return "吃草";
       }
    }
    Animal.prototype.name = "小狗";
    Animal.prototype.eat = function(){
       return "吃肉";
    }
    //console.log( Animal.prototype ); //object
    var animal = new Animal();
    alert( animal.name ); //小羊
    alert( animal.eat() );//吃草
</script>

原型方法可以复用：
    var arr1 =new Array(1,2,3,4,5);
    var arr2 = new Array(11,2,3,4,5);
    Array.prototype.sum = function(){
       var s = 0;
       for( var i = 0 ; i < this.length ; i++ ){
           s += this[i];
       }
       return s;
    }
    alert( arr1.sum() );
    alert( arr2.sum() );   方法复用

原型模式的执行流程 ： 
首先在实例中查找，如果找到了 ，就返回  
如果实例中没有，就去原型中查找，如果找到就返回

7、继承方式一、通过改变父类的执行环境

<script type="text/javascript">
    function Father(){
       this.money = 999999;
       this.eat = function(){
           return "爱吃肉";
       }
    }

    function Son(){
       this.parent = Father;//将父类构造函数赋值给子类属性
       this.parent();//通过调用子类属性   完成继承父类
       //删除实例属性
       delete this.parent;
    }
    var son = new Son();
    console.log( son );
</script>
8、继承方式二   call继承

call()方法：
第一个参数 ： 指向子类对象的this 第二个参数：要继承的父类的属性  第二个参数的个数不固定
9、继承方式三  Apply

apply()  方法：
第一个参数 ：  指向子类对象的this    第二个参数 ： 数组 （数组中的数据就是继承的父类的属性   个数不固定）   第二个参数可以写成 arguments
