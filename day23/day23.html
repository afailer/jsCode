2、对象

什么是对象?   
对象是一个整体，对外提供一些功能        Array  String   Math  Date  正则 window  document  event....  系统对象 

对象的特点 ： 
     一切具有本质特征和行为的物质
     一切具有属性和方法的事物    Array ：  length    push()  pop()   
对象特性 ： 属性 和  方法     对象.属性   对象.方法()

描述一辆汽车 ：
     属性： 颜色  型号  品牌  价格     
     方法： 行驶   启动  
     p标签 ： 
          属性 ： 宽 高 
          方法  ：    自定义

数据类型 ：
     所有的基本类型都没有属性和方法
     var num = 90;
     num.index = 1;
     alert(num.index);
     所有的对象类型都有属性和方法
     
方法和函数的关系：
     方法 属于 函数    函数 包含  方法     

3、面向对象

面对对象  ：  是一种编程思想   
Array  String。。。   使用这些对象时  就是一种面向对象的编程思想
面向过程和面向对象 ：
     面向过程  ： c语言     程序按照某种步骤  从上到下执行    
     面向对象 ：  java  php   c#

例如：
一条公路  ：长20000米    不行速度  50米/分钟    走完这条路需要多久？

面向过程：
     长度/速度 = 时间

面向对象：
     公路 ： 属性  长度      
     人 ：   属性  速度     功能 行走   
     在人对象的行走方法中完成  ： return  公路.length / 人.速度

面向对象的优缺点：
     缺点： 项目开发周期长   成本高
     优点：
          功能独立  互不干扰
          变量私有化  防止全局变量污染

类和对象的关系 ：  
     类是对象的抽象
     对象是类的实例
     类 ---  构造函数

2、 var obj = new Object();

特点 ： 一次只能定义一个对象       如果多个同类的对象重复定义时，代码重复性大

5、工厂模式 （设计模式）

优点 ：解决了同类对象创建时 代码重复的问题 
缺点 ： 对象是通过函数调用的方式创建的，不符合真正对象的创建规范（一般对象是通过new关键字创建出来的）
       不能确定某个对象属于哪一个类   
      语法：   对象 instanceof  构造函数
6、构造函数（类）

构造函数中的属性叫做实例属性
构造函数中的方法叫做实例方法
为了和普通函数区分，一般构造函数命名成大驼峰个   Array   String  Math  Date
构造函数中的this指向构造函数new出来的 对象  

优点 ：
     能够确定某个对象属于哪一个类
     通过new关键字创建对象  符合对象的创建规范 
缺点 ：
     同类对象的相同方法会被重新创建   

7、原型对象

写在构造函数外面的属性叫做原型属性
写在构造函数外面的方法叫做原型方法
优点 ：
     解决了同类对象方法重复创建的问题
缺点 ：所有对象的属性值相同

8、混合 

属性写成实例属性
方法写成原型方法
